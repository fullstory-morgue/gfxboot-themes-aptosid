Įkrovos parametrai

There are two types of boot options available. First, there are options that affect the installer. Second, there are kernel options. Some of the more common options are:

a) diegimo parametrai

  • įdiegti -- pasirinkite diegimo šaltinį
  • tinklo parametrai -- tinklo parinktys
  • vnc parametrai -- parametrai diegimui per VNC

b) branduolio parametrai

  • splash -- daro įtaką pristatymo ekrano elgsenai
  • apm -- perjungti maitinimo valdymą
  • acpi -- sudėtingesnė konfigūracija ir energijos sąsaja
  • ide -- kontroliuoja IDE posistemį
