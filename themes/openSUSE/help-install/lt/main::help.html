Naudojimasis pagalbos sistema

Paleidyklės nuotolinė pagalba labai priklauso nuo konteksto. Pateikia informaciją apie meniu punktą ar ieško informacijos apie parinktį, ties kuria yra žymeklis, jei keičiate įkrovos parinktis.

Judėjimo klavišai

  • Viršutinė rodyklė: paryškinti ankstesnę nuorodą
  • Apatinė rodyklė: paryškinti kitą nuorodą
  • Kairioji rodyklė, Naikinti: grįžti į ankstesnę temą
  • Dešinioji rodyklė, Įvesti (Enter), Tarpas: sekti nuorodą
  • Psl Aukštyn (Page Up): eiti į puslapio pabaigą
  • Psl Žemyn (Page Down): eiti į puslapio pabaigą
  • Pradžia (Home): eiti į puslapio pradžią
  • Pabaiga (End): eiti į puslapio pabaigą
  • Gr (Esc): palikti pagalbą
