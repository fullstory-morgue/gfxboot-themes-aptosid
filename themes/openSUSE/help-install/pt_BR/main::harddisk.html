Inicializar do disco rígido

Selecione Iniciar o SO instalado para iniciar o sistema instalado no disco rígido local. Esse sistema deve estar instalado corretamente, pois apenas o MBR (Master Boot Record) no primeiro disco rígido é iniciado. O ID do dispositivo do primeiro disco rígido é fornecido pela BIOS do computador.

Utilize esta opção se você tiver esquecido de remover o CD ou o DVD da unidade e desejar iniciar o computador do disco rígido.
