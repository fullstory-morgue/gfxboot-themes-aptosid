Opțiuni de boot

Există două tipuri de opțiuni de boot disponibile: opțiunile ce afectează instalatorul și opțiunile kernel. Câteva dintre opțiunile uzuale sunt:

a) opțiuni instalator

  • instalează -- selectați o sursă de instalare
  • opțiuni rețea -- opțiunile rețea
  • opțiuni vnc -- opțiuni pentru instalarea via VNC

b) opțiuni kernel

  • splash -- influențează comportamentul ecranului de întâmpinare
  • apm -- comută gestiunea alimentării
  • acpi -- configurare avansată și interfață alimentare
  • ide -- controlează subsistemul IDE
