Selecție mod video

Apăsați @@@fkey_video@@@ pentru a obține lista modurilor video suportate de placa dvs. grafică. Modul de afișare cel mai înalt suportat de monitor va fi preselectat.

Este posibil ca monitorul dvs. să nu poată fi detectat automat. În acest caz, selectați manual modul dorit.

Dacă la instalare pe sistemul dvs. apar probleme cu placa grafică, o soluție posibilă este mod text.
