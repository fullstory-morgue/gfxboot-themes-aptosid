Справка загрузчика

Добро пожаловать @@@product@@@

Используйте это меню для выбора необходимой функции. Если у вас проблемы с управлением в данной системе помощи, нажмите@@@fkey_help@@@ чтобы войти в описание системы помощи. Главные функции в этом меню:

Загрузка с жёсткого диска: Этот выбор ничего ничего не сделает с системой. Просто будет запущена ранне установленная система.

Установка: Этот режим установки работает на большинстве компьютеров. Если у вас зависает система при загрузке или у вас проблемы с обнаружением аппаратных средств, таких как контроллеры диска или сетевые платы, попробуйте одну из следующих установочных параметров.

Установка - ACPI выключено: Многие проданные компьюетры имеют неполную или неверную реализацию ACPI. Этот выбор отключает поддержку ACPI в ядре, однако многие особенности влияющие на быстродействие все равно включены, напримерe DMA для IDE жестких дисков.

Установка - локальный APIC выключен: Если обычная установка не получается, возможно ваше оборудование не поддерживает APIC. Если это так, используйте этот параметр для установки без поддержки APIC.

Установка - безопасные параметры настройки: Если у вас не получается с Установка, этот выбор может помочь вам.

Спасательная система: Этот загрузочный образ запускает маленькую Linux систмеу в RAM. Это полезно если ваша система запускает не правильно. После загрузки этой системы, войдите как root.

Проверка программно-аппаратных средств: Запускает инструмент проверки BIOS, который проверяет ACPI и другие части вашего BIOS.

Тест памяти: Тест памяти более полезней, чем просто проверка новых модулей памяти. Это стрессовая проверка большей части вашей компьютерной системы и может выявить проблемы в оборудовании.

Параметры загрузки: Параметры загрузки могут полностью изменять поведение вашей системы. Это настройки ядра.

@@@fkey_help@@@ Help: Зависит от содержания. Будут показаны разные экраны в зависимости от активного элемента экрана загрузки. Также есть описание этой системы помощи.

@@@fkey_lang@@@: Установить язык и раскладку клавиатуры используемые загрузчиком.

@@@fkey_video@@@ Видео режим: Выберите необходимое разрешение экрана, используемое при установке. Если у вас есть проблемы с графической установкой, текстовый режим может быть поможет вам.

@@@fkey_inst@@@ Источник установки: Выберите источник установки.

@@@fkey_dud@@@ Драйвер обновления: Для очень новых машин, системе может понадобиться Драйвер обновления.
