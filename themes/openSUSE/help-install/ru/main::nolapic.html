Установка - локальный APIC выключен

Обычно не возникают проблемы с APIC (Усовершенствованный интерфейс конфигурирования системы и управления энергопитанием). Но если вы обнаружили некоторые, попробуйте этот параметр для загрузки без поддержки APIC.

Не путайте это с ACPI (Усовершенствованный интерфейс конфигурирования системы и управления энергопитанием).
