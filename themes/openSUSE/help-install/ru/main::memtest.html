Тест памяти

Включенный параметр Тест памяти предоставляет хорошую возможность для стрессового теста оборудования системы. Главная цель обнаружить неисправные RAM, но кроме этого происходит нагрузка остальных частей системы.

Если не обнаружено ошибок, нет гарантии, что память хорошая, однако большинство дефектов памяти были бы обнаружены.
