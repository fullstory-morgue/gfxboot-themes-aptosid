Установка - ACPI выключено

Современное оборудование, обычно использует APIC (Усовершенствованный интерфейс конфигурирования системы и управления энергопитанием) для управления прерываниями. ACPI полностью заменяет старую APM систему.

Выберите Установка - ACPI выключено если есть проблемы с ядром во время загрузки. Известные проблемы с машинами, которые имеют проблемы с AC:

  • ядро приостанавливается при загрузке
  • PCI платы не обнаружены или неверно инициализированы

Вы можете попробовать использовать Проверка программно-аппаратных средств параметр загрузки и посмотреть результат проверки ACPI.
